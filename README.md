# Sistema de Gestão de Reservas de Hotel

## Autor
- Yelifer Ferreira

## Descrição Geral
Este projeto consiste na criação de um sistema simples em Python para gerir 
reservas, quartos e clientes de um hotel. O foco principal é a organização 
do projeto com Git e GitHub.

## Funcionalidades Principais
- Registo e listagem de clientes
- Gestão de quartos (disponíveis/ocupados)
- Criar e cancelar reservas
